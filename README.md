# GigaHR

</div>

Умный Telegram-бот, который помогает соискателям и компаниям с HR-задачами.
<br>
В его основе встроен искусственный интеллект GigaChat, специально обученный для работы в сфере HR.

### Возможности бота
* Показ вакансий - бот выводит актуальный список вакансий компании.
* Запись на собеседование - соискатель может оставить заявку прямо в чате.
* HR-консультации от ИИ - GigaChat отвечает на вопросы и помогает подготовиться к собеседованию.
* Уведомления - бот сообщает, когда открывается вакансия или появляется возможность записаться на собеседование, а также оповещает сотрудников о кандидатах. 
* Режим для сотрудников - при вводе специальной команды бот переключается в режим работы с HR-сотрудниками: позволяет просматривать заявки, менять статусы кандидатов и управлять датами собеседований.


## Как запустить
1. **Установите зависимости**
<br>Перейдите в терминале в корень проекта и выполните команду
   ```
   pip install -r requirements.txt
   ```

2. **Создайте базу данных PostgreSQL.**
<br>Используйте файл **gigahr_database.sql** для создания таблиц.
<br>Заполните таблицы данными. Для этого можете использовать файлы с префиксом **insert_**.

3. [Создайте бота в Telegram](https://www.t.me/BotFather), чтобы получить его токен.
4. Также понадобится токен для использования GigaChat. [Инструкция по получению.](https://developers.sber.ru/docs/ru/gigachat/quickstart/ind-create-project)
5. Откройте файл **config.json**, который находится в корне проекта, и вставьте нужные данные в соответствующие поля.

6. **Запустите бота**
<br>Выполните в терминале из корневой папки проекта
    ```
    python -m src.main
    ```
 
**Программа написана на Python 3.11**
